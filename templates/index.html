Project Purpose & Interest

Applying our knowledge gained over the last three years in the BSIT program, we aim to narrow our focus to software development utilizing cloud technologies. From beginning to end and everything in between, we are seeking to put our gained knowledge and further interest to implement a real project from start to finish. We are eager to learn along the way and look forward to working with our project sponsor.

Development

Implementation and web services start with the development process in mind from architecture of the system and interfaces to the implementation details.

Security

Develop secure system by following the process with security at the forefront.

Configuration

Customizable and configurable interactions when interacting with a rich API.

Responsive

Client interfaces must be adaptable automatically for user experience as services are accessed via an array of screen shapes and sizes.

Loose Coupling

OOP appropriate software patterns and implementations. Apply appropriate knowledge and practices to produce the more advantages decoupled system.
